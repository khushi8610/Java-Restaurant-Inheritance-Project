# Restaurant Menu System (Java)

This project is a simple Java program that demonstrates Object-Oriented Programming (OOP) concepts such as inheritance and method overriding using a restaurant example.

## Project Description
The program simulates a food delivery application where different types of restaurants have their own menus. A base class `Restaurant` is created, and two subclasses `VegRestaurant` and `NonVegRestaurant` extend it to provide their specific menus.

## Features
- Base class `Restaurant`
- Variable: Restaurant Name
- Method: `showMenu()`
- Subclass `VegRestaurant` with its own menu
- Subclass `NonVegRestaurant` with its own menu
- Demonstrates **Inheritance and Method Overriding**

## Technologies Used
- Java
- Object-Oriented Programming (OOP)

## Project Structure
Restaurant
├── VegRestaurant
├── NonVegRestaurant
└── Main

## Example Output
Veg Menu: Paneer Butter Masala, Veg Biryani, Dal Makhani
Non-Veg Menu: Chicken Biryani, Butter Chicken, Fish Curry
